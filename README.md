# Kawsay Tours

Página web para informarte sobre los principales lugares turísticos del Perú, con la posibilidad de hacer tu reservación en línea.

## Estructura del proyecto

```
kawsay-tour/
├── index.html    Estructura principal de la página
├── style.css     Estilos (tema oscuro con detalles dorados)
├── script.js     Interactividad: menú móvil, buscador, carrusel, contadores, scroll y botón "subir"
└── img/          Imágenes usadas en el sitio (agrégalas aquí)
```

## Cómo verlo localmente

Solo abre `index.html` en tu navegador, o usa un servidor local, por ejemplo:

```bash
npx serve .
```

## Imágenes necesarias

El sitio referencia imágenes en la carpeta `img/` (por ejemplo `img/peru.jpg`, `img/cusco.jpg`, `img/machu.jpg`, etc., una por cada departamento y sección). Agrega tus propias imágenes con esos mismos nombres para que se muestren correctamente.

## Publicar en GitHub Pages

1. Sube este proyecto a un repositorio de GitHub.
2. Ve a **Settings > Pages**.
3. Selecciona la rama `main` y la carpeta raíz (`/`).
4. Guarda y espera unos minutos: tu sitio quedará disponible en `https://tu-usuario.github.io/kawsay-tour/`.
